## 页面布局

```
┌─────────────────────────────────────────┐
│            Header: 卡片库               │
├─────────────────────────────────────────┤
│    [行为]       [规律]       [灵感]      │  ← 一级 Tab，underline 指示器
├─────────────────────────────────────────┤
│  ┌──────────┐  ┌──────────┐            │
│  │正确的事 4 │  │错误的事 3 │            │  ← 二级 pill 切换
│  └──────────┘  └──────────┘            │
├─────────────────────────────────────────┤
│  [分类▾]  [排序▾]                       │  ← 筛选栏
├─────────────────────────────────────────┤
│                                         │
│  卡片列表                                │
│                                         │
└─────────────────────────────────────────┘
│              [+] FAB                     │
```

## 一级 Tab

- 样式：沿用现有 `.card-lib-tabs` 风格，改为 3 列等宽
- 带数量 badge（该一级类目下所有活跃卡片总数）
- 切换时二级默认选中第一项

## 二级切换

- 样式：pill/segment 形态，背景色区分选中态
- 行为 Tab 下：「正确的事 (n)」「错误的事 (n)」
- 规律 Tab 下：「正向 (n)」「负向 (n)」
- 灵感 Tab 下：「正向 (n)」「负向 (n)」
- 选中态颜色跟随语义：正向=绿系，负向=红/橙系

## 卡片统一骨架

三行结构，左侧 4px 色条：

```
┌─ 色条 ─────────────────────────────────────┐
│  [标题]                     [badge/标签]    │  ← 行1：主信息
│  [icon 分类] · [关联/来源]                  │  ← 行2：元信息，灰色小字
│  [指标A] · [指标B]                          │  ← 行3：数据，灰色小字
└────────────────────────────────────────────┘
```

### 各卡片填充规则

| 卡片类型 | 色条 | 标题 | Badge | 元信息 | 数据行 |
|----------|------|------|-------|--------|--------|
| 正确的事 | 分类色 | name | 置顶/归档 | 分类 | 执行N次 · 成功率N% |
| 错误的事 | #FF6B35 | name | 红线 | 分类 | 权重N · 关联N条规律 |
| 正向规律 | #36D399 | law_desc | — | 分类 · 关联行为 | 触发N次 · 权重N |
| 负向规律 | #F87272 | law_desc | — | 分类 · 关联行为 | 触发N次 · 权重N |
| 正向灵感 | #FBBF24 | desc | — | 分类 · 来源 | 日期 |
| 负向灵感 | #A78BFA | desc | — | 分类 · 来源 | 日期 |

## 交互规则

| 手势 | 行为 | 适用范围 |
|------|------|----------|
| 点击 | 进入详情页/面板 | 全部卡片 |
| 长按 / 右键 | 弹出操作菜单 | 全部卡片 |
| 滑动 | 无（移除） | — |

## 筛选栏

统一两个 select：

- **分类**：全部分类 / 各分类选项（所有 Tab 共用）
- **排序**：因 Tab 而异
  - 正确的事：主观权重 / 执行次数 / 成功率
  - 错误的事：主观权重
  - 规律：主观权重 / 触发次数
  - 灵感：创建时间

## 色彩语义

| 语义 | 色值 | 用途 |

| 灵感正向 | #FBBF24 | 正向灵感色条 |
| 灵感负向 | #A78BFA | 负向灵感色条 |
