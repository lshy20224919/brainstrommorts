# 方法论进化器

一个帮助个人进步的方法论沉淀工具，旨在提升做事成功率。

## 技术栈

- **前端**: uni-app (微信小程序)
- **后端**: Node.js + Express
- **数据库**: MySQL

## 项目结构

```
methodology-evolver/
├── server/                 # 后端服务
│   ├── src/
│   │   ├── config/        # 配置文件
│   │   ├── middleware/    # 中间件
│   │   ├── routes/        # API路由
│   │   ├── utils/         # 工具函数
│   │   └── index.js       # 入口文件
│   ├── sql/              # 数据库脚本
│   └── package.json
└── client/               # 前端小程序
```

## 快速开始

### 1. 初始化数据库

```bash
# 创建数据库
mysql -u root -p < sql/init.sql

# 或者使用命令行
mysql -u root -p
source sql/init.sql
```

### 2. 配置环境变量

```bash
cp .env.example .env
# 编辑 .env 文件，填入数据库配置
```

### 3. 启动后端服务

```bash
npm install
npm run dev
```

服务启动后访问 http://localhost:3000/health

## 数据模型

- **争取的事 (action_right)**: 用户认可的做事准则，先定义，多次复用执行
- **打卡/事件**: 争取的事在某次具体场景中的执行记录
- **规律**: 从成功/失败中提炼的正向/负向逻辑，与争取的事并列为核心起点
- **SOP**: 多个争取的事+规律的标准化流程模板

## 核心流程

```
争取的事 + 规律 → 按规律执行 → 打卡 → 复盘 → 进化
```

> 争取的事和规律是并列的两个起点，核心是"按规律做正确的事"
